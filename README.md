# FileBox —— 安卓通用文件查看与编辑器

FileBox 是一款基于 Android SAF 的文件浏览与文本编辑工具，支持多种文本格式与语法高亮，专注轻量、好用、免费。

## 主要特性
- 文件浏览与选择：基于 SAF，支持应用私有与外部目录访问
- 新建/重命名/删除：支持常见文件操作
- 新建文本与 JSON：便捷创建并立即打开编辑
- 语法高亮与编辑：常用文本高亮、查找替换、撤销重做
- 列表刷新机制：新建后自动刷新、返回自动刷新、工具栏一键刷新
- APK.1打开：支持跳转系统安装器（受系统策略限制）
- 主题与样式：马卡龙粉主题、统一配色
- 适配 Android 12 - Android 15
